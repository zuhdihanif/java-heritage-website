Folder asset : Folder asset berisi berbagai asset yang digunakan dalam program, seperti css, js, image, dan vendor yang berisi bootstrap.
index.html : Merupakan file html yang menjadi landing page dari program.
book.html : Merupakan file halaman html yang menjadi page untuk booking form.
book.php : Merupakan file php yang digunakan melakukan koneksi ke database dan mengambil data dari form.
